Bootcamp Project

YelpCamp

A full-stack web app where users can add and rate campgrounds. Users are able to make updates to their own campgrounds and reviews. Integrates with mapbox to show geolocation of each campground. Built using Express, EJS templating, MongoDB and Nodejs.

Technologies used include:

  • HTML
  • CSS
  • JavaScript
  • MongoDB
  • Node.js

A fun project

NASA Astronomy Picture of the Day

Fun HTML, CSS, and JavaScript site that queries NASA's backend API for the Astronomy Picture of the Day. Users are able to also use a date picker to view historical pictures of the day. I've always enjoyed astronomy, and found this to be a really fun project. I got my hands dirty with TailwindCSS, and had fun learning the basics of the framework.

Technologies used include:

  • HTML
  • TailwindCSS
  • JavaScript
